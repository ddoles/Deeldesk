Navigation System Demo

This demo shows the responsive navigation system for Deeldesk.ai.

Desktop Navigation (1024px+)

On desktop, you'll see a persistent left sidebar with all navigation items. The sidebar is always visible and provides quick access to all sections.

  • Persistent sidebar (240px wide)
  • Active state indicators
  • Badge counts for items
  • User menu in footer

Mobile Navigation (<1024px)

On mobile and tablet, the sidebar is hidden by default and accessible via hamburger menu. A bottom navigation bar provides quick access to main sections.

  • Hamburger menu in header
  • Bottom navigation bar
  • FAB for "New Proposal"
  • Touch-friendly targets (44px+)
